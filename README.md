## Installation Client Side

- Step 1: Clone the repository
- Step 2: Run the command line "npm install"
- Step 3: Run the command line "npm run dev"

## Installation Server Side
- Step 1: Clone the repository
- Step 2: Run the command line "npm install"
- Step 3: Run the command line "npm run watch"
